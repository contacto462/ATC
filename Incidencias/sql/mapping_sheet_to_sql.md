# Mapeo rápido Sheets -> SQL

## Registro
- `A (ODT)` -> `registro.odt`
- `B (Fecha)` -> `registro.fecha_registro`
- `C (Puesto)` -> `registro.puesto`
- `D (Cliente/Sucursal)` -> `registro.cliente`
- `E (Problema/Tipo)` -> `registro.problema`
- `F (Derivación)` -> `registro.derivacion`
- `G (Observación/Detalle)` -> `registro.detalle_problema` / `registro.observacion`
- `H (Técnico)` -> `registro.tecnicos`
- `I (Estado)` -> `registro.estado`
- `J (Días ejecución)` -> `registro.dias_ejecucion`
- `K (Fecha cierre)` -> `registro.fecha_cierre`
- `L/M (Fechas derivación)` -> `registro.fecha_derivacion_area` / `registro.fecha_derivacion_tecnico`
- `N (Dirección)` -> `registro.direccion`
- `O (Observación final)` -> `registro.observacion_final`
- `S (Prioridad)` -> `registro.prioridad`
- `T (Materiales)` -> `registro.materiales`
- `U (Acompañante)` -> `registro.acompanante`
- `V (% avance)` -> `registro.porcentaje_avance`
- `X/Y/Z (Imágenes)` -> `registro.foto_1` / `registro.foto_2` / `registro.foto_3`

## BBDD
- Hoja completa -> `bbdd_clientes`

## ODT / Administración

- `Administración` -> `administracion_odt`

## Rendición
- Hoja `Rendición` -> `rendiciones`

## Tareas
- Hoja `Tareas` -> `tareas`

## Contactos de emergencia
- Hoja `ContactosEmergencia` -> `contactos_emergencia`
